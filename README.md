# tour-of-heroes vue
replica of tour-of-heroes angular project in vue js with vue router and and vuex

## Project setup
```
npm install
```

### Compiles and hot-reloads for development

### Compiles and minifies for production

### Run your unit tests

```
npm run test:unit
```

### Run your end-to-end tests

```
npm run test:e2e
```
